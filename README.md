# Flask Task API

A production-style Flask REST API with PostgreSQL and DynamoDB.

## Project Setup

### 1. Prerequisites
- Python 3.10+
- Docker & Docker Compose

### 2. Installation
Clone the repository and set up the virtual environment:
```bash
git clone <your-repo-url>
cd flask-task-api
python -m venv venv
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt
